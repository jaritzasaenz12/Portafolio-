Diego Esquivel

Fecha 2020
Locación Cartago, Costa Rica
Cliente Diego Andrés Esquivel

La marca personal “Diego Esquivel” nace al ver la necesidad de tener una imagen que represente el trabajo profesional como fotógrafo por lo que se realiza una construcción de un logotipo, monograma minimalista partiendo de las las palabras claves que definen la marca: “ seguridad y confianza”

Se utiliza el elemento gráfico de un círculo que representa el lente de una cámara siguiendo un estilo minimalista, para el desarrollo tipográfico se agrega la inical del nombre (Diego) “D” minúscula para adaptarla a la forma del lente y se retocan los remates para conseguir la unión entre la letra y el símbolo.
